## Project Purpose

This repository contains the complete source code of the CodeXtremeOS website, designed to:

- Present the operating system features
- Provide technical documentation
- Offer updates and news
- Show screenshots and demonstrations
- Provide guides and tools

## Local Development

Prerequisites:
- Node.js v20+
- pnpm v9+

Installation steps:

1. Clone repository:
```bash
git clone https://github.com/CodeF1ow/the-street-edge.git
cd the-street-edge
```
2. Install dependencies:
```bash
pnpm install
```
3. Start development server:
```bash
pnpm run dev
```
4. Build for production:
```bash
pnpm run build
```
